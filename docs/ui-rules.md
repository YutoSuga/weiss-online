# ui-rules.md

# UI操作仕様

## 概要

本ドキュメントは、プレイヤーがプレイ画面上で行う操作、および各操作に対するUIの振る舞いを定義する。

本仕様はPC版を対象とする。

---

# 基本方針

* 直感的な操作を目指す
* 実際のカードゲームに近い操作感を再現する
* 不要な確認ダイアログは極力表示しない
* 誤操作しにくいUIを目指す

---

# カード操作

## 選択

### 左クリック

カードを選択状態にする。

選択されたカードは枠線や影などで強調表示する。

---

## 移動

カードをドラッグ&ドロップして移動する。

移動可能なエリアへドロップした場合のみ移動を確定する。

---

## 拡大表示

ダブルクリックでカードを拡大表示する。

拡大表示中は背景を暗くし、もう一度ダブルクリックまたはEscキーで閉じる。

---

# 山札

## ドロー

山札をクリックすると1枚ドローする。

ドローしたカードは手札へ移動する。

# 手札

カードはドラッグで各エリアへ移動できる。

手札内では並び替え可能。

---

# 舞台

カードは空いている枠へ配置できる。

カードの入れ替えはドラッグ操作で行う。

---

# ストック

カードを追加すると最前面へ重なる。

クリックしても順番は変更しない。

---

# クロック

カードを追加すると右方向へ重なる。

---

# レベル

カードは順番に配置する。

---

# 控え室

カードは重ねて管理する。

枚数のみ常時表示する。

---

# 思い出

カードは横向きで管理する。

---

# クライマックス置場

1枚のみ配置可能。

2枚目を置こうとした場合は配置できない。

---

# ショートカットキー

| キー       | 操作       |
| -------- | -------- |
| Esc      | 拡大表示を閉じる |
| Delete   | （未定）     |
| Ctrl + Z | （未定）     |

---

# エフェクト

## カード移動

200〜300ms程度のアニメーションで移動する。

---

## ドロー

山札から手札へ滑らかに移動する。

---

## 選択

カードを少し拡大表示する。

# 操作ポリシー

* 操作回数はできるだけ少なくする。
* よく使う操作はワンクリックで実行できるようにする。
* 対戦テンポを妨げないUIを目指す。
* 実際のカードゲーム経験者が違和感なく操作できることを重視する。
