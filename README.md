weiss-online

概要

weiss-online は、ブラウザ上で友人同士が対戦できるカードゲームプラットフォームです。

本プロジェクトでは、実際のカードゲームに近い操作感を目指し、ドラッグ＆ドロップによるカード操作、オンライン対戦、デッキ管理などを実装します。

このプロジェクトは個人利用・学習目的で開発を進めます。

⸻

開発目標

* ブラウザだけで遊べる
* PC・スマホ両対応
* 友人同士でオンライン対戦
* 直感的なカード操作
* 拡張しやすい設計

プレイ画面

各プレイヤーは以下のエリアを持ちます。

* 手札
* クロック
* レベル
* クライマックス
* 舞台（前列3・後列2）
* ストック
* 山札
* 控え室
* 思い出

画面レイアウトは実際のプレイ感を重視し、相手側・自分側を180°回転した配置を基本とします。
